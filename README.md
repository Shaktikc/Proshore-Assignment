# ![Monicle Emoticon](/public/favicon.ico) React Repository Search

### A repository search application using [Github's repository search API]. The results list can be **sorted** by best match, stars, and forks.

## Table of contents

- [Setup](#setup)
- [Technologies](#technologies)
- [Getting Started with Create React App](#Getting Started with Create React App)

## Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```

## Technologies

Project is created with:

- React
- Javascript
- Redux
- React DOM
- Ui Chakra
